# StatusLinker

## Feature Highlights

- Allows you to pick a <status> from any status in the game.

## Installation

- `https://raw.githubusercontent.com/Catphinaud/DalamudPlugins/refs/heads/main/pluginmaster.json`

## Usage

- Use the command `/slink` to open the plugin window.
- Select a status from the dropdown menu.
- Put <status> in the chat and there you go.
